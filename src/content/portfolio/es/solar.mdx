Solar es un simulador interactivo de gravedad y sistema solar construido para el navegador. Permite explorar cuerpos celestes, observar mecánicas orbitales, controlar el tiempo de simulación y lanzar asteroides con parámetros configurables como radio, densidad, velocidad, azimut y elevación.

El proyecto combina una experiencia visual 3D con paneles de control para inspeccionar planetas, cambiar la cámara y experimentar con el comportamiento gravitacional del sistema.
