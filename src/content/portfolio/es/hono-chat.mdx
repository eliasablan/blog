Hono Chat es una aplicación de chat en tiempo real con salas basadas en URL. El frontend está construido con Next.js y React (TypeScript), y el backend con Hono. Utiliza WebSockets para mensajería bidireccional y PostgreSQL para persistencia de datos. Lista para producción.
